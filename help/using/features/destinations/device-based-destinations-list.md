# [!DNL Audience Manager] [!UICONTROL Device-based Destinations]

이 페이지에는 현재 항목이 포함되어 있습니다 [!DNL Audience Manager] [장치 기반 대상](/help/using/features/destinations/add-device-based-destinations.md) 전용. in [!DNL Audience Manager], [!UICONTROL device-based destinations] 라는 호출도 [!UICONTROL server-to-server destinations]. 통합에 대한 자세한 내용은 다음을 참조하십시오 [Adobe 교환](https://exchange.adobe.com/experiencecloud.html) 목록 작성(사용 가능한 경우) 또는 [!DNL Adobe] 컨설턴트 또는 고객 지원 .


* **마지막 업데이트 날짜** = 이 항목에 대한 업데이트가 있는 마지막 날짜를 표시합니다 [!UICONTROL device-based destination].
* **업데이트** = 여부를 나타냅니다. [!DNL Audience Manager] 데이터를 이 데이터 세트에 실시간 또는 배치(1일)로 보냅니다 [!UICONTROL destination].
* **세그먼트 해제** = 세그먼테이션 해제 는 장치 프로필을 [!UICONTROL segments]. 에서 장치 프로필을 제거하는 기능 [!UICONTROL segment] 을(를) 만드는 데 사용되는 장치 옵션에 따라 다릅니다. [!UICONTROL Profile Merge Rule]. 세그먼테이션 해제는 또한 [!DNL GDPR] 삭제 요청. 읽기 [프로필 병합 규칙 및 장치 세그먼테이션 해제 프로세스](/help/using/features/profile-merge-rules/merge-rule-unsegment.md) 및 [세그먼테이션 해제 기능이 있는 Audience Manager 파트너](/help/using/overview/data-security-and-privacy/data-privacy-requests.md#aam-partners-with-unsegmentation). [!DNL Audience Manager] 세그먼트 해제 정보를 로 보냅니다. [!UICONTROL destinations] 표시 **예** 아래의 제품에서 사용할 수 있습니다.
* **수신 [!DNL IAB TCF] 동의 신호** = [!DNL Audience Manager Plug-in for IAB TCF] 다음을 수행할 수 있습니다. [!DNL IAB TC] 사용자의 개인 정보 보호 선택 사항을 준수하면서 활성화 파트너에 문자열입니다. 읽기 [IAB TCF용 Audience Manager 플러그인](/help/using/overview/data-security-and-privacy/aam-iab-plugin.md).
* **[!DNL ICDS]** = 이 열에는 를 지원하는 파트너가 나열됩니다 [!UICONTROL Instant Cross-Device Suppression]. 자세한 내용 [즉각적인 장치 간 억제](/help/using/features/profile-merge-rules/instant-cross-device-suppression.md).
