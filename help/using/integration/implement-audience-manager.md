# Implementing Audience Manager {#implementing-audience-manager}

This section outlines and explains the processes related to getting started with the Audience Manager data management platform ([!DNL DMP]). 이 섹션은 비즈니스 팀, 프로젝트 관리자 및 기술 관리자가 Audience Manager 구현 프로세스를 이해할 수 있도록 설계되었습니다. Audience Manager를 시작하려면 데이터 수집 요구 사항에 따라 약 6 주가 걸릴 수 있습니다.

Adobe 구현 기술은 신규 고객과 컨설팅 파트너 관계를 구축하는 데 도움이 됩니다. 이 프로세스는 다음과 같이 설계되었습니다.

* 귀사의 비즈니스 요구 사항을 발견하고 이해합니다.
* 이러한 요구를 해결하기 위한 실행 가능한 계획을 수립합니다.
* 고유한 요구 사항 또는 사용 사례를 충족하는 사용자 정의 솔루션을 개발할 수 있습니다.
* 독점 데이터를 가져와 Audience Manager에서 사용할 수 있도록 하십시오.

Adobe 파트너 솔루션 및 계정 관리 팀은 구현 과정 전이나 후에, 그리고 구현 후에도 귀하와 긴밀하게 협력할 것입니다.

Audience Manager는 설정 및 구현에 대한 단계별 접근 방식을 수행합니다.

## Build, Test, and Train Phase {#build-test-train-phase}

빌드, 테스트 및 트레이닝 단계에서 지정된 파트너 솔루션 리드를 통해 데이터 수집 전략 및 프로토타입을 검토합니다.

<!-- c_build_test_train.xml -->

Your data collection strategy will undergo end-to-end [!DNL QA] testing. 파트너 솔루션은 발견한 버그를 추적하고 시스템 엔지니어를 통해 문제를 해결합니다. 고객 트레이닝은 이러한 다른 노력과 동시에 시작할 수 있습니다.

다음 표에서는 이 단계 중에 발생하는 주요 활동에 대해 설명합니다.

| 활동 | 목적/설명 | 참가자 |
|---|---|---|
| 데이터 수집 전략 준비 | Adobe 기술 팀과 협력하여 비즈니스 요구 사항을 충족하는 데이터 수집 계획 구축 | 비즈니스 팀과 기술 팀 |
| 배포 및 테스트 코드 | 스테이징 환경에서 제안된 솔루션을 테스트하고 브라우저 간 테스트 수행 | 기술 팀 |
| 기능 확인 및 버그 해결 | 결과 검사 및 전달, 버그 해결, 다시 테스트 | 기술 팀 |

* 수락 및 승인.

## Code Implementation {#code-implementation}

Though the deployment process may seem complex, the code implementation is as simple as adding a few lines of [!DNL JavaScript] adjacent to the closing `</body>` tag of your website.

<!-- c_code.xml -->

### 배포

The Audience Manager code snippet calls [!DNL Akamai] to download the business rules set up previously in the user interface. 또한 클라이언트 브라우저는 이 정보를 캐시하므로 페이지 및 서버 로드 시간을 줄이는 데 도움이 됩니다. Adobe의 코드 및 데이터 수집 방법은 인벤토리 전반에 걸쳐 사용자 경험을 유지하기 위해 고안되었습니다.

### 참가자

파트너 솔루션은 기술 팀과 직접 협력하여 코드를 배포하고 최종 문제를 해결하며 다른 요구 사항을 충족할 수 있습니다.

## Post-Implementation Support {#post-implementation-support}

Adobe의 협업 노력은 최종 배포와 함께 멈추지 않습니다. 구현이 완료되면 계정 관리 팀이 인계받습니다.

<!-- c_post_implement_support.xml -->

제품 구현 프로세스가 완료된 후 계정 관리자는 지속적인 지원 및 컨설팅 서비스를 제공합니다. 계정 관리자와 정기적인 회의를 가질 수 있습니다. 이러한 미팅을 통해 Audience Manager에서 최대한의 사용 및 가치를 얻을 수 있습니다.

Contact us [here](https://www.adobe.com/products/audiencemanager.html) for more information and to get started with Audience Manager.
