# [!UICONTROL Audience Optimization Reports] {#audience-optimization-reports}

[!UICONTROL Audience Optimization Reports]은 데이터 시각화 방법을 사용하여 Audience Manager 계정의 [!UICONTROL destinations]에 대한 정보를 반환합니다. 각 보고서에서 거의 모든 데이터 포인트를 클릭하여 해당 항목에 대한 세부 정보를 반환할 수 있습니다. 이러한 [!UICONTROL Audience Optimization] 인사이트는 광고 및 게시 채널에서 여러 사용 사례에 적용할 수 있지만 기본적으로 사용할 수는 없습니다. 시작하려면 Audience Manager 컨설턴트에게 문의하십시오.
