# 비공개 데이터 피드 {#private-data-feeds}

비공개 데이터 피드는 공급자가 구매자의 데이터에 대한 액세스를 제한할 수 있는 옵션입니다. 데이터 제공자와 구매자는 개인 데이터 피드를 만들고 구독하기 전에 이 정보를 검토해야 합니다.

<!-- c_marketplace_privatefeed.xml -->

## Private Data Feeds for Providers {#private-data-feeds-providers}

제공업체에서는 데이터 피드가 공개 또는 비공개일 수 있습니다. 비공개 데이터 피드를 사용하면 데이터 판매자 이름을 포함하여 구매자의 데이터 액세스를 제한할 수 있습니다. 특별 거래, 할인 또는 개인 정보 및 액세스 제어 기능이 중요한 경우 개인 데이터 피드를 만들어 제공할 수 있습니다. 비공개 데이터 피드를 사용하면 구매자 요청을 검토하고 승인할 수 있습니다. 요청을 승인하면 피드가 구매자에게 공개 데이터 피드처럼 보입니다. You can view and manage all your feeds in **[!UICONTROL Audience Marketplace > My Shared Data]**. 아래에서 보듯이, 이 유형의 피드는 상태 열에 &quot;비공개&quot; 로 표시됩니다.

![](assets/my_shared_data.png)

## Private Data Feeds for Buyers {#private-data-feeds-for-buyers}

## How to set up the Sharing Relationship between Data Provider and Data Buyer {#set-up-sharing-relationship}

### 1 단계 - 활성화 - 데이터 공급자 및 데이터 구매자

이 프로세스의 첫 번째 단계는 Adobe 컨설팅 또는 고객 지원 센터의 개입이 필요합니다. 데이터 제공업체 및 데이터 구매자는 Adobe 컨설팅 또는 고객 지원 센터에 연락하여 지원을 요청해야 합니다.

### 2 단계 - 데이터 공급자 - 새 데이터 소스 만들기

Audience Manager 계정에서 다음 항목으로 새 쿠키 데이터 소스를 만듭니다.

* **Audience Manager ID** 를 인바운드 키로
* **공유 활성화** 옵션이 선택되었습니다.

![](assets/create-datasource.png)

**저장을** 클릭하면 **트레이트 스토리지 &gt; 타사 데이터에 새 하위 폴더가 자동으로 만들어집니다**.

![](assets/folder-structure.png)

### 3 단계 - 데이터 공급자 - 공유할 트레이트를 식별합니다.

이 단계에서는 파트너와 공유할 트레이트를 식별합니다. 새 특성을 만들거나 기존 트레이트를 편집할 수 있습니다. 어떤 경우에서든 트레이트가 필요합니다.

* To be associated with the data source you created as part 2.
* 타사 데이터 아래에서 새로 만든 하위 폴더에 저장됩니다.

Read more about [creating traits](/help/using/features/traits/create-onboarded-rule-based-traits.md) and [editing traits](/help/using/features/traits/manage-trait-rules.md#edit-trait).

### 4 단계 - 데이터 공급자 - 데이터 피드 만들기

데이터 피드를 만들어 데이터 구매자와 트레이트를 공유할 수 있습니다. Refer to [Create a Public or Private Data Feed](/help/using/features/audience-marketplace/marketplace-data-providers/marketplace-create-manage-feeds.md) for instructions on how to create a data feed.

>[!IMPORTANT]
>
>설정에서 개인 옵션을 선택합니다. 이 필드를 public로 설정하면 Audience Marketplace 고객이 피드에 가입할 수 있습니다.

![](assets/create-data-feed.png)

### 5 단계 - 데이터 구매자 - 액세스 요청

Go to **Audience Marketplace &gt; Marketplace**. 이전 단계에서 데이터 공급자가 생성한 데이터 피드를 검색합니다. Click **Request Access**. 데이터 공급자 측의 지정된 연락처에 이메일 알림이 전송됩니다. See also, [Subscribe to a Private Data Feed](/help/using/features/audience-marketplace/marketplace-data-buyers/marketplace-manage-subscriptions.md#subscript-private-data-feed).

### 6 단계 - 데이터 공급자 - 액세스 권한 부여

**Audience Marketplace &gt; 내 공유 데이터로** 이동하고 4 단계에서 만든 피드를 검색합니다. Click into the new access request and click **Allow Access** to approve the request. See also, [Review, Approve, or Reject Private Feed Requests](/help/using/features/audience-marketplace/marketplace-data-providers/marketplace-create-manage-feeds.md#manage-private-requests).

### 7 단계 - 데이터 구매자 - 가입 활성화

After the data provider grants access to the data feed, you can see the feed in your account in **Audience Marketplace &gt; Marketplace**. Review the details, turn the Subscription button ON, and click **Review &amp; Subscribe**. See [Storage for Subscribed Data Feeds](/help/using/features/audience-marketplace/marketplace-data-buyers/marketplace-manage-subscriptions.md#find-subscribed-data-fee) for information on where to find the 3rd party traits.

이러한 트레이트는 데이터 공급자의 계정에서만 편집할 수 있습니다.
