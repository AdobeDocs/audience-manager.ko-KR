# 2019 설명서 업데이트 {#documentation-updates}

2019년에 출시된 신규 또는 수정된 설명서 목록. Experience Cloud 릴리스 노트에서 다루지 않을 수 있는 Audience Manager 설명서의 부분적 업데이트를 포함합니다.

[!DNL Audience Manager] 및 기타 [!DNL Experience Cloud] 솔루션에 대한 변경 사항에 대해서는 [Experience Cloud 릴리스 노트](https://docs.adobe.com/content/help/ko-KR/release-notes/experience-cloud/current.html)를 참조하십시오. 이전 릴리스 노트가 필요하면 [이전 연도](../docs-updates/docs-2018.md) 섹션을 참조하십시오.

## 2019년 11월 {#aam-november-updates}

| 주제 | 설명 |
|--- |----|
| [CCPA(캘리포니아 소비자 개인 정보 보호법) 지원 및 개인 정보 보호 설명서 정비](../overview/data-security-and-privacy/data-privacy.md) | 2020년 1월 1일부터 시작되는 [캘리포니아 소비자 개인 정보 보호법(CPA)](https://www.caprivacy.org/about)은 캘리포니아 주민들에게 개인 정보와 관련된 새로운 권리를 제공하고 캘리포니아에서 사업을 수행하는 특정 업체에 데이터 보호 책임을 부과할 예정입니다.<br> <br> Audience Manager은 데이터 액세스 및 삭제 요청에 대한  [Adobe Experience Platform 개인정보 보호 ](https://www.adobe.io/apis/experienceplatform/home/services/privacy-service.html) 서비스와 같은 개인 정보 보호 툴을 통해 개인정보 보호 규정에 따른 의무를 준수하도록 도와줍니다. <br><br> 선언된 ID( [예: CRM ID)](../overview/data-security-and-privacy/data-privacy-requests.md#opt-out-requests) 에서 옵트아웃하는 것을 포함하도록 현재 옵트아웃 관리 프로세스를 업데이트했습니다. 선언된 ID로 옵트아웃하는 경우, 선언된 ID와 마지막으로 연결된 장치가 Audience Manager 데이터 수집에서 옵트아웃됩니다. 이제 옵트아웃 요청도 일괄 처리에서 실시간으로 이 기능을 지원하는 [대상 파트너](../overview/data-security-and-privacy/assets/AAM-Partners-October2019.xlsx)에게 세그먼트 취소 요청을 보냅니다. <br><br> 또한, Adobe는 앞서 언급한 규정을 준수하는 데 필요한 정보를  [쉽게 찾을 수 있도록 ](../overview/data-security-and-privacy/data-security.md)데이터 보안 [](../overview/data-security-and-privacy/data-privacy.md),  [데이터 개인정보](../overview/data-security-and-privacy/data-governance.md)  보호및데이터정부 설명서를 다시 설계했습니다. |
