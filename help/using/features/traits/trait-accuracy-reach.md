# 정확성 및 도달 범위 {#accuracy-and-reach}

알고리즘 트레이트에서 정확도와 도달 사이의 관계를 설명합니다.

<!-- c_accuracy_reach.xml -->

## 정확성과 도달 범위:정보

알고리즘 트레이트를 사용할 때 정확도와 도달 사이의 관계를 이해하는 것이 중요합니다. 정확성은 사용자가 기준선과 얼마나 유사한지를 반영하는 점수가 높은 값으로 표시됩니다. 정확도 비율 범위는 0(가장 정확하지 않음)부터 1(가장 정확함)까지입니다. 도달 범위는 트레이트에 포함할 고유 사용자 수를 나타내는 값일 뿐입니다. 도달 범위와 정확성은 역방향으로 관련되어 있습니다. 정확한 트레이트는 더 적은 사용자에게 도달하고 더 큰 도달 범위를 갖는 트레이트는 정확성이 떨어집니다. 다음 이미지는 이 개념을 보여줍니다.

![](assets/Reach_v_Accuracy.png)

## 대상 크기에 영향을 주는 정확성 및 도달 범위

비즈니스 목표를 통해 알고리즘 트레이트를 사용할 때 정확도에 대한 올바른 의사 결정을 내리고 도달 범위를 확대할 수 있습니다. 정확도가 목표인 경우 트레이트의 모집단은 모델 실행에서 증가 또는 감소할 수 있습니다. 모집단 변경은 각 평가 기간 동안 알고리즘을 결정하는 결과의 결과입니다. 경우에 따라 알고리즘이 처리 주기 중에 더 적합한 사용자를 찾고, 그 밖의 사용자에게는 더 적은 수의 사용자를 찾을 수 있습니다. 결과는 이전 모델 실행 이후 발생한 모델 및 새 방문자 및 트레이트 자격을 만드는 데 사용되는 기준 데이터에 의해 결정됩니다. 반면에 도달 범위 작업 시 사용자 인구 수는 일정하게 유지됩니다. 예를 들어 10,000명의 사용자에게 도달하려면 알고리즘이 각 모델 실행에 대해 항상 해당 숫자를 히트하도록 합니다.

## 정확도와 도달 범위에 대한 일반적인 사용 사례

정확성 또는 도달 범위에 대한 초점은 특정 세그먼트를 사용하여 이루고자 하는 내용에 따라 다릅니다. 다음 표를 사용하면 트레이트를 만들 때 정확도와 도달 범위를 평가할 수 있습니다.

| 트레이트 결정 선호도 | 찾는 데 도움이 됩니다. |
|---|---|
| **정확도** | 모델의 기본 고객과 유사한 사용자. 특정 대상에게 도달하려는 경우 타깃팅된 캠페인에 유용합니다. |
| **도달** | 각 데이터 실행에 대한 특정 사용자 수입니다. 특정 크기의 대상에게 도달하려는 경우 브랜드 캠페인에 유용합니다. |
