# Audience Manager 구현 {#implementing-audience-manager}

이 섹션에서는 Audience Manager 데이터 관리 플랫폼([!DNL DMP])을 시작하는 것과 관련된 프로세스에 대해 설명하고 설명합니다. 이 섹션에서는 비즈니스 팀, 프로젝트 관리자 및 기술 관리자가 Audience Manager 구현 프로세스를 이해할 수 있도록 돕습니다. 데이터 수집 요구에 따라 Audience Manager을 시작하는 데 약 6주에서 3개월이 걸릴 수 있습니다.

Adobe의 구현 기술을 통해 신규 고객과의 컨설팅 파트너 관계를 구축할 수 있습니다. 이 프로세스는

* 비즈니스 요구 사항을 발견하고 파악합니다.
* 이러한 요구를 해결할 수 있는 실행 가능한 계획을 제시합니다.
* 고유한 요구 사항 또는 사용 사례를 충족하는 맞춤형 솔루션 개발
* 개인 데이터를 Audience Manager에서 가져와 사용할 수 있도록 해야 합니다.

Adobe의 파트너 솔루션 및 계정 관리 팀은 구현 과정 전후로 고객과 긴밀하게 협력할 것입니다.

Audience Manager은 단계적 접근 방식으로 설정하고 이행한다.

## 빌드, 테스트 및 교육 단계 {#build-test-train-phase}

구축, 테스트 및 교육 단계 동안 지정된 파트너 솔루션 리드를 통해 데이터 수집 전략 및 프로토타입을 검토할 수 있습니다.

<!-- c_build_test_train.xml -->

데이터 수집 전략은 종단간 [!DNL QA] 테스트를 거칩니다. 파트너 솔루션은 발견한 버그를 추적하고 시스템 엔지니어와 문제 해결 방법을 조율합니다. 고객 트레이닝은 이러한 다른 노력과 동시에 시작할 수 있습니다.

다음 표에서는 이 단계 동안 수행되는 주요 활동에 대해 설명합니다.

| 활동 | 목적/설명 | 참가자 |
|---|---|---|
| 데이터 수집 전략 준비 | Adobe 기술 팀과 협력하여 비즈니스 요구 사항을 충족하는 데이터 수집 계획 수립 | 비즈니스 및 기술 팀 |
| 코드 배포 및 테스트 | 스테이징 환경에서 제안된 솔루션을 테스트하고 브라우저 간 테스트를 수행합니다. | 기술 팀 |
| 기능 확인 및 버그 해결 | 테스트 및 커뮤니케이션, 버그 해결, 재테스트 | 기술 팀 |
| 사용자 교육 | Audience Manager 기능, 툴 및 보고서에 대한 교육 및 이해 | 비즈니스 팀 |

이 단계의 결과물에는 다음이 포함될 수 있습니다.

* 완료된 데이터 수집 계획
* 엔드 투 엔드 [!DNL QA] 테스트;
* Audience Manager 유저 인터페이스 기능에 대한 기본 지침
* 동의 및 로그아웃.

## 코드 구현 {#code-implementation}

배포 프로세스가 복잡해 보일 수 있지만 코드 구현은 웹 사이트의 닫는 `</body>` 태그에 인접한 [!DNL JavaScript]의 몇 줄을 추가하는 것만큼 간단합니다.

<!-- c_code.xml -->

### 배포

Audience Manager 코드 조각은 [!DNL Akamai]을 호출하여 사용자 인터페이스에 이전에 설정한 비즈니스 규칙을 다운로드합니다. 또한 클라이언트 브라우저는 이 정보를 캐시하여 페이지 및 서버 로드 시간을 줄입니다. Adobe 코드 및 데이터 수집 방법은 전체 인벤토리에서 사용자 경험을 유지 관리할 수 있도록 고안되었습니다.

### 참가자

파트너 솔루션은 기술 팀과 직접 협력하여 코드를 배포하고 최종 문제를 해결하며 기타 요구 사항을 충족할 수 있습니다.

## 사후 구현 지원 {#post-implementation-support}

공동 작업을 통한 최종 배포는 물론 구현이 완료되면 Account Management 팀이 인계합니다.

<!-- c_post_implement_support.xml -->

계정 관리자는 제품 구현 프로세스가 완료된 후 지속적인 지원 및 컨설팅 서비스를 제공합니다. 계정 관리자와 정기적인 회의를 가질 수 있습니다. 이러한 회의를 통해 Audience Manager의 사용 및 가치를 최대화합니다.

자세한 내용과 Audience Manager을 시작하려면 [여기](https://www.adobe.com/products/audiencemanager.html)로 문의하십시오.
