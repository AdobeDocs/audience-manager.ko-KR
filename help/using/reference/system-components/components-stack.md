# Audience Manager 시스템의 주요 구성 요소{#key-components-in-the-audience-manager-system}

Audience Manager은 시스템과 프로세스를 네 가지 주요 범주로 그룹화합니다.태그 관리, 데이터 수집, 데이터 조직 및 데이터 실행 가능.
