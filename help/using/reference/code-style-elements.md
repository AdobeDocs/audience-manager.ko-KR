# 코드 및 텍스트 요소에 대한 스타일 규칙{#style-conventions-for-code-and-text-elements}

이러한 요소는 도움말 설명서 전체에서 사용되는 코드 옵션과 변수를 식별합니다. 일반적으로 코드나 데이터 파일에 이러한 기호나 스타일 요소를 포함하지는 않습니다. 시각적 표시만 있습니다.

  <tr> 
   <td colname="col1"> <p> <code> <i>variable</i> </code> </p> </td> 
   <td colname="col2"> <p>변수가 <i>기울임체</i>에 나타납니다. 변수 이름에 적절한 값을 대체합니다. </p> <p>명확하게 하기 위해 때로 변수와 기타 코드 요소가 &lt; &gt; brackets 사이에 표시됩니다. 코드에 이러한 기호를 사용할 필요는 없습니다. </p> </td> 
  </tr>

  </tr>

  </tr>
