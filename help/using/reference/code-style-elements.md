# 코드 및 텍스트 요소에 대한 스타일 규칙{#style-conventions-for-code-and-text-elements}

이러한 요소는 도움말 설명서 전체에서 사용되는 코드 옵션 및 변수를 식별합니다. 일반적으로 이러한 기호나 스타일 요소는 코드나 데이터 파일에 포함되지 않습니다. 시각적인 지표일 뿐이죠.
