# 비공개 데이터 피드 {#private-data-feeds}

개인 데이터 피드는 공급자가 구매자의 데이터에 대한 액세스를 제한할 수 있는 옵션입니다. 데이터 제공업체 및 구매자는 개인 데이터 피드를 만들고 구독하기 전에 이 정보를 검토해야 합니다.

<!-- c_marketplace_privatefeed.xml -->

## 공급자에 대한 개인 데이터 피드 {#private-data-feeds-providers}

공급자의 경우 데이터 피드는 공개 또는 비공개일 수 있습니다. 개인 데이터 피드를 사용하면 데이터 판매자의 이름을 포함하여 구매자의 데이터 액세스를 제한할 수 있습니다. 특별 거래, 할인 또는 개인 정보 및 액세스 제어가 중요한 경우 비공개 데이터 피드를 만들 수 있습니다. 개인 데이터 피드를 사용하면 구매자 요청을 검토하고 승인할 수 있습니다. 요청을 승인하면 피드는 구매자에 대한 공개 데이터 피드처럼 보입니다. 모든 피드를 보고 관리할 수 **[!UICONTROL Audience Marketplace > My Shared Data]**&#x200B;있습니다. 아래와 같이 이 유형의 피드는 상태 열에 "비공개"로 표시됩니다.

![](assets/my_shared_data.png)

### 피드 요청 관리

에서 개인 데이터 피드의 이름을 클릭하면 여러 탭이 포함된 페이지로 [!UICONTROL My Shared Data] 이동합니다. 탭을 클릭하여 개인 데이터 피드 요청을 관리합니다.

![](assets/shared_data_tabs.png)

## 구매자를 위한 개인 데이터 피드 {#private-data-feeds-for-buyers}

구매자는 개인 데이터 피드를 다른 [오퍼와](../../features/audience-marketplace/marketplace-data-buyers/marketplace-data-buyers.md#about-marketplace) 마찬가지로 Marketplace에 표시합니다. 하지만 이 경우 피드 목록에는 트레이트, 고유 사용자 및 사용자 겹침 요약 정보가 표시되지 않습니다. 또한 데이터 판매자는 목록의 [!UICONTROL Provider] 열에 이름을 표시하거나 숨길 수 [!UICONTROL Marketplace] 있습니다. 판매자가 구독 요청을 승인하면 비공개 피드의 모든 데이터를 사용할 수 있습니다(공개 피드와 마찬가지로 작동). 아래 [!UICONTROL Marketplace] 예에는 구매자로 사용할 수 있는 세 가지 다른 피드 유형이 나와 있습니다.

## 데이터 제공자와 데이터 구매자 간의 공유 관계를 설정하는 방법 {#set-up-sharing-relationship}

### 1단계 - 지원 - 데이터 공급자 및 데이터 구매자

이 과정의 첫 번째 단계는 Adobe 컨설팅 또는 고객 지원 센터의 개입을 필요로 합니다. 데이터 제공업체 및 데이터 구매자는 Adobe 컨설팅 또는 고객 지원 센터에 문의하여 지원을 요청해야 합니다.

### 2단계 - 데이터 공급자 - 새 데이터 소스 만들기

Audience Manager 계정에서 다음을 사용하여 새 쿠키 데이터 소스를 만듭니다.

* **Audience Manager** ID를 인바운드 키로
* 공유 **활성화** 옵션이 선택되어 있습니다.

![](assets/create-datasource.png)

저장을 클릭하면 **트레이트**&#x200B;스토리지 &gt; 타사 데이터에 새 하위 **폴더가 자동으로 생성됩니다**.

![](assets/folder-structure.png)

### 3단계 - 데이터 공급자 - 공유할 트레이트 식별

이 단계에서는 파트너와 공유할 트레이트를 식별합니다. 새 트레이트를 만들거나 기존 트레이트를 편집할 수 있습니다. 어떤 경우든 트레이트가 필요합니다.

* 단계 2의 일부로 생성한 데이터 소스와 연결됩니다.
* 새로 만든 하위 폴더의 타사 데이터 아래에 저장됩니다.

트레이트 [만들기 및](/help/using/features/traits/create-onboarded-rule-based-traits.md) 트레이트 [](/help/using/features/traits/manage-trait-rules.md#edit-trait)편집에 대한 자세한 내용을 살펴보십시오.

### 4단계 - 데이터 공급자 - 데이터 피드 만들기

그런 다음 데이터 피드를 만들어 데이터 구매자와 트레이트를 공유합니다. 데이터 피드를 [만드는 방법에 대한 지침은 공개](/help/using/features/audience-marketplace/marketplace-data-providers/marketplace-create-manage-feeds.md) 또는 비공개 데이터 피드 만들기를 참조하십시오.

>[!IMPORTANT]
>
>설정에서 개인 옵션을 선택합니다. 이 필드를 공개로 설정하면 모든 Audience Marketplace 고객이 피드에 가입할 수 있습니다.

![](assets/create-data-feed.png)

### 5단계 - 데이터 구매자 - 액세스 요청

Audience Marketplace **&gt; Marketplace로 이동합니다**. 이전 단계에서 데이터 공급자가 만든 데이터 피드를 검색합니다. 액세스 **요청을 클릭합니다**. 이제 데이터 제공업체측에서 지정한 담당자에게 전자 메일 알림을 수신하게 됩니다. 비공개 [데이터 피드 가입을 참조하십시오](/help/using/features/audience-marketplace/marketplace-data-buyers/marketplace-manage-subscriptions.md#subscript-private-data-feed).

### 6단계 - 데이터 공급자 - 액세스 권한 부여

Audience **Marketplace &gt; 내 공유** 데이터로 이동하여 4단계에서 만든 피드를 검색합니다. 새 액세스 요청을 클릭하고 액세스 **허용을** 클릭하여 요청을 승인합니다. 비공개 피드 [요청 검토, 승인 또는 거부를 참조하십시오](/help/using/features/audience-marketplace/marketplace-data-providers/marketplace-create-manage-feeds.md#manage-private-requests).

### 7단계 - 데이터 구매자 - 구독 켜기

데이터 공급자가 데이터 피드에 대한 액세스 권한을 부여하면 Audience Marketplace &gt; Marketplace에서 계정에서 피드를 볼 **수 있습니다**. 세부 사항을 검토하고 [구독] 단추를 [설정]한 다음 [검토 및 **구독]을 클릭합니다**. 타사 [트레이트를](/help/using/features/audience-marketplace/marketplace-data-buyers/marketplace-manage-subscriptions.md#find-subscribed-data-fee) 찾을 위치에 대한 자세한 내용은 구독된 데이터 피드의 저장소를 참조하십시오.

이러한 트레이트는 데이터 공급자의 계정에서만 편집할 수 있습니다.
