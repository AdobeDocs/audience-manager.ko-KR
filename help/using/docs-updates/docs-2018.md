# 2018 설명서 업데이트 {#documentation-updates}

2018년 및 이전 해에 릴리스된 신규 또는 수정된 설명서 목록. Experience Cloud 릴리스 노트에서 다루지 않을 수 있는 Audience Manager 설명서의 부분적 업데이트를 포함합니다.
