# 세그먼트 목록 보기 {#segments-list-view}

대시보드는 대상을 관리하기 위한 중앙 작업 공간입니다. [!UICONTROL Segments]

기본 [!UICONTROL Segments] 페이지에는 다음과 같은 유용한 기능과 도구가 포함되어 있습니다.

* 새 세그먼트를 만듭니다.
* 세그먼트 편집 및 삭제
* 기존 세그먼트를 복제(복제)합니다.
* 정렬 가능한 열이 있는 표에서 모든 세그먼트를 볼 수 있습니다.
* 세그먼트 저장소 관리
* 이름별로 세그먼트를 검색합니다.
